VISUAL STUDIO CODE

¿Qué es el visual studio code?

El vscode es el mejor editor de código del mundo.

Funciones del vscode

* Intellisense: Nos ayuda a autocompletar un código, y nos da imformación de las funciones
* Extensibilidad: Permite agregar extenciones para dar mucha más funcionalidad
* Control de Versiones Integrados: Nos facilita el seguimiento del código, porque nos permite integrar sistemas de versiones.
* Debugging: Tiene herramientas que nos permiten analizar linea por linea el código y ver que está sucediendo en tiempo real, mientras se ejecuta el código.
* Personalización: Nos permite personalizar nuestra area de trabajo a nuestro gusto.
